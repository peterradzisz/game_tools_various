## How to Use These Links

### Option A: Download all manually
1. Click each link
2. Download the Windows `.exe` or `.zip` file
3. Save to `D:\BG2_ModStaging\downloads\`
4. Run `Install-All.bat` from the game directory

### Option B: Use the files in this repo
The `mods/` folder contains 13 of 14 mod files (Warlock skipped due to 150MB GitHub limit). Just copy them to your staging directory and install.

### Option C: Automated PowerShell download
The `Install-BG2EE-Mods.ps1` script can auto-download GitHub-hosted mods. For Artisan mods, it opens the browser for manual download.
